# Description
This is a trivia web application. It utilizes a trivia api to fetch random questions about random topics.
A user gets to answer the random questions and find out their score. 

Include the following npm:

npm install ejs mongoose mongodb dotenv axios cors express

# API link
https://opentdb.com/api.php?amount=10&category=9&type=multiple
https://opentdb.com/api.php?amount=100&category=9&type=multiple
